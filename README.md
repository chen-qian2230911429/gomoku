# 五子棋 AI (Gomoku AI)

这是一个用 C 语言实现的五子棋 AI 程序，包含禁手判断、棋型评估和极小极大搜索算法。

## 功能特性
- ✅ 支持人机对战（PVE）和人人对战（PVP）
- ✅ 完整的禁手规则判断（三三、四四、长连）
- ✅ 基于棋型评估的 AI 搜索算法，15s内实现八层搜索
- ✅ 模块化设计，代码结构清晰

## 编译运行
1. 确保安装了 GCC 编译器
2. 执行编译脚本：
   ```bash
   gcc -O3 -flto gomoku.c gomoku_base.c gomoku_evaluate.c gomoku_forbidden_move.c -o gomoku.exe
   ```

## 具体实现

- 为了保证随时终止的正确性，采用迭代加深，目前为8层，可以进行修改。
- 对多种复合棋形进行判断与赋值。
- TT表hash比对进行剪枝
- 利用极小极大算法，采用 $\alpha-\beta$ 剪枝。
- 选取已有棋子点位附近点位进行搜索，进行预先估价排序，设计后悔增选，保证下子的正确性。
